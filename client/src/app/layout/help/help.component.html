Ajuda para Utilização do Sistema de Processamento

Abaixo segue instruções para realizar os procedimentos disponiveis nesta plataforma. Os processamentos são para fins de pesquisas e estão em densenvilvimento, para relatar erros e problemas entrar em contato.


Processamento CNN

Existe duas opções de exames para o Processamento CNN disponível atualmente, "VFC" e "COVID-19". Em "VFC" apartir de um arquivo .TXT contendo os valores de VFC por linha, pode-se realizar o processamento da CNN para descobir problemas cardiacos como: Hipertermsão, Arritimia e Saudável.

Em "VFC" apartir de um arquivo .TXT contendo os valores de VFC por linha, pode-se realizar o processamento da CNN para descobir problemas cardiacos como: Hipertermsão, Arritimia e Saudável.

Em "COVID-19" apartir de uma imagem em .jpeg/.jpg/.png, pode-se realizar o processamento da CNN para descobir problemas pulmonares: COVID-19, Outras e Saudável.


Como utilizar o módulo Processamento CNN


  1. Clique em "tipo de diagnóstico" para selecionar o tipo de exame.
  2. Clique em arquivo de entrada para escolher uma imagem/arquivo para realizar o processamento.
    • Clique em "teste" caso não possua imagem/arquivo para processar.
    • Clique em "selecionar" para seleciona-la.
    • Pode-se mudar de imagem/arquivo teste clicando em Próxima/Anterior.
  3. Após selecionar uma imagem/arquivo clique em "avaliar".

Pseudo-Coloração

Para a pseudo-coloração de raio-x utiliza-se uma imagem em formato .jpeg/.jpg/.png. Primeiramente é realizado um pre-processamento de equilização acumulada do histograma e posteriormente é realizada a pseudo-coloração.


Como utilizar o módulo Pseudo-Coloração


  1. Clique em "arquivo de entrada" para escolher uma imagem para realizar a Pseudo-Coloração.
    • Clique em "teste" caso não possua imagem/arquivo para processar.
    • Clique em "selecionar" para seleciona-la.
    • Pode-se mudar de imagem/arquivo teste clicando em Próxima/Anterior.
  2. Após selecionar uma imagem clique em "avaliar".
